Traitement automatique

Format de fichier requis
Colonnes obligatoires
  • PN : Numéro de pièce (Part Number)
  • Project : Nom du projet
Colonnes optionnelles
  • Price : Prix du composant
  • Supplier : Fournisseur
  • Description : Description du composant
  • Status : Statut (A, D, 0, X)
Règles de format
  • Première ligne = en-têtes de colonnes
  • Format Excel (.xlsx ou .xls)
  • Taille maximum : 100MB
  • Encodage : UTF-8 recommandé
Logique de traitement
Étapes de traitement

  • Suppression des lignes vides
  • Normalisation des formats
  • Conversion en majuscules
  • Suppression des espaces superflus

  • Correspondance PN + Project
  • Récupération du statut
  • Identification des nouveaux composants

  • Status D → Mise à jour vers X (déprécié)
  • Status 0 → Création ligne pour vérification
  • Status NaN → Nouveau composant potentiel
  • Status X → Ignoré (déjà ancien)
Dépannage

Solution :

  • Vérifiez que le fichier est au format .xlsx ou .xls
  • Évitez les fichiers .csv ou .txt
  • Assurez-vous que le fichier n'est pas corrompu

Solution :

  • Ajoutez les colonnes PN et Project
  • Vérifiez l'orthographe des en-têtes
  • Assurez-vous que la première ligne contient les en-têtes

Solution :

  • Vérifiez la taille du fichier (max 100MB)
  • Réduisez le nombre de lignes si nécessaire
  • Actualisez la page et réessayez
